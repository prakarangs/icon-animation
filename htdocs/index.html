Animation with Mo.js

1. TranslateY with Mo.js

var bounceProgress = mojs.easing.bounce.out(progress);

iconHeart.style.transform =
'translateY(' + 200*bounceProgress + 'px)';

Mo.js use Tween method to animate style of the element. While using (element).style.transform = 'value'; we can animate like CSS3.

Progress is increasing number from 0 to 1.

200*Progress
This basically means, we will translate the position of our element bit by bit until we reach 200px. (without easing)

2. scale3d transformation

var bounceProgress = mojs.easing.bounce.out(progress);
icon.style.transform =
'scale3d(' + bounceProgress + ',' + bounceProgress + ',1)';

We can also pass in diffrent parameters into CSS3 animation functions.

scale3d(x,y,z);
